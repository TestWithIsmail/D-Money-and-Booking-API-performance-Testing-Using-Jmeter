## Jmeter Testing

### Project Summary: In this project, I work with two websites.

- restful-booker.herokuapp.com: Here, I can log in as an admin; I can create bookings and search bookings.
- dmoney.roadtocareer.net: In this website, agents can deposit money to the users, customers can send money to each customer, customers can pay to merchants, and customers can also withdraw money from an agent.

### Technology I have used
- Jmeter
- Postman

### How to run?
1. Open Command Prompt and write jmeter
2. Open .jmx file in jmeter
